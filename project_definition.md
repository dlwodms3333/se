# 프로젝트 정의서

## 1. 프로젝트 명칭
- 조직 내 협업을 위한 클라우드 파일 공유 시스템 개발

## 2. 개발 시스템 명칭
- Mini Drive

## 3. 개발 시스템 설명
- 조직 내부 사용자를 위한 클라우드 기반 파일 관리 시스템
- 파일 업로드, 다운로드, 공유 기능 제공
- 폴더 구조를 통해 파일 정리 가능
- 검색, 버전 관리, 권한 제어 기능 지원

## 4. 핵심 기능
- 파일 업로드 및 다운로드
- 폴더 생성, 이름 변경, 삭제
- 파일 이동 및 정리
- 사용자 간 파일 공유
- 보기 / 수정 / 댓글 권한 설정
- 외부 링크 공유
- 파일 검색
- 파일 버전 관리
- 사용자 로그인 및 관리자 기능
- 보안 기능

## 5. 유사 소프트웨어 조사

### Google Drive
- 파일 저장 및 공유 중심 서비스
- 폴더 관리, 링크 공유, 권한 설정 가능
- 검색 기능과 버전 관리 기능 지원
- 본 프로젝트의 파일 관리 구조 참고 가능

### Notion
- 문서 작성 및 협업 중심 서비스
- 페이지 기반 정보 정리 기능 제공
- 댓글, 공유, 권한 관리 기능 지원
- 협업 기능 측면에서 참고 가능
